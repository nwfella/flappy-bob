# Flappy Bob 🧽🐦

A SpongeBob SquarePants themed Flappy Bird clone — built from scratch with vanilla HTML, CSS, and JavaScript Canvas.

**Play it live:** https://nwfella.github.io/flappy-bob

## Features

- 🎨 **Custom SpongeBob character** — hand-drawn on canvas with pores, shirt, tie, pants, eyes, nose, smile, and teeth
- 🪸 **Coral pipe obstacles** — textured with Bikini Bottom vibes
- 🌊 **Animated bubbles** — with wobble physics in the background
- 🏖️ **Sandy sea floor** — with moving sand dimple details
- 📱 **Mobile friendly** — touch, tap, and keyboard controls
- 🏆 **Best score** — persisted via localStorage
- 🔄 **Full game loop** — start screen → gameplay → game over → replay

## How to Play

- **Desktop:** Press Space or Click to flap
- **Mobile:** Tap the screen to flap
- Navigate through the coral pipes to score points
- Don't hit the pipes, floor, or ceiling!

## Tech Stack

- **Canvas 2D API** — all rendering done programmatically
- **Vanilla JS** — no frameworks, no dependencies
- **Tailwind CSS** — overlay UI styling (CDN)
- **localStorage** — persistent high scores

## About

Built by [nwfella](https://github.com/nwfella) as a portfolio project demonstrating canvas game development, collision detection, animation loops, and responsive mobile-first design patterns.

Inspired by the classic Flappy Bird by Dong Nguyen and the one and only SpongeBob SquarePants.
